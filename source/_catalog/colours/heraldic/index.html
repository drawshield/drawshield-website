These are shades that DrawShield uses for the main heraldic colours by default. Sadly, the origin of the choice of shade is lost is the mists of time but I'm sure I had a good reason for using these colours, I just can't remember what it was. Following that are the other main palettes supported by DrawShield that probably have a more robust pedigree. If you have no other preference then I would probably recommend the Wikipedia palette.

Note that if a colour is not shown in an alternate palette then the corresponding shade from the default DrawShield palette will be used instead.
